# Frontend - Intelli-Credit React App

## Overview
Modern React.js frontend for credit decisioning interface.

## Setup

```bash
cd frontend
npm install
```

## Run

```bash
npm start
```

Frontend runs on: http://localhost:3000

## Pages

- **Dashboard** - Landing page with features
- **New Application** - Credit application form
- **Results** - Credit analysis results
- **About** - Project information
